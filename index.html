История Происхождения

История создания GitHub началась с появления в 2005 году системы управления версиями Git. Разработчики начали применять эту систему в своей работе. Идея создания GitHub пришла Крису Ванстрасу и Тому Престону-Вернеру во время митапа Ruby-разработчиков в октябре 2007 года в Сан-Франциско. Они планировали, что будущий проект должен стать местом, где разработчики могли бы легко делиться кодом и изучать программы друг друга. В конце 2007 года они приступили к созданию собственного проекта — по субботам разработчики устраивали мозговой штурм, а в свободное время писали код будущей платформы. Престон-Вернер взял на себя разработку пользовательского интерфейса GitHub, а Ванстрас занялся разработкой самого приложения на Ruby on Rails. В начале 2008 года разработчики запустили бета-версию платформы и разослали приглашения друзьям и знакомым. Первые участники начали пользоваться GitHub, размещая на площадке свои рабочие коды. Публичный запуск GitHub состоялся 10 апреля 2008 года. К этому моменту проект содержал 20 тысяч репозиториев, а количество пользователей составило более двух тысяч.

О GitHub

GitHub — это сервис для совместной разработки и хостинга проектов. С его помощью над кодом проекта может работать неограниченное количество программистов из любых точек мира.

Функции

Хостинг

он позволяет хранить проекты удалённо на сервере и работать с ними из любой точки мира.

Контроль версий

Все изменения в коде можно отследить, поэтому в командной разработке это незаменимая вещь.
